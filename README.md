# Aprendizaje-no-Supervisado
Este trabajo aplica Machine Learning No Supervisado (PCA y K-Means) sobre el dataset "Graduate Admission" para segmentar aspirantes a posgrado. Tras estandarizar datos y eliminar redundancias, se redujo la dimensionalidad, logrando clústeres un 28% más definidos mediante PCA, identificando dos perfiles claros de éxito académico.
